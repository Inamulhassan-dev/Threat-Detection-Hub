# Start Here (Windows)

Use these files in this order:

1. Double-click setup.bat (first time only)
2. Double-click start.bat (every time you want to run)
3. Open http://localhost:5000
4. Login with:
   - username: admin
   - password: admin123
5. Double-click stop.bat to stop
